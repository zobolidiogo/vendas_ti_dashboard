# Dashboard Interativo: Análise de Vendas TI

![Power BI](https://img.shields.io/badge/Power%20BI-F2C811?style=for-the-badge&logo=powerbi&logoColor=black)

Dashboard interativo desenvolvido com foco em storytelling e tomada de decisão para análise de vendas de produtos tecnológicos (acessórios e eletrônicos).

---

## O Desafio

Criar um dashboard que apresentasse, de forma clara e objetiva, insights sobre vendas de produtos de TI, permitindo identificar rapidamente padrões, produtos de destaque e oportunidades de melhoria através de storytelling efetivo.

---

## A Solução

Dashboard com análise completa de vendas, aplicando princípios de design (Heurísticas de Nielsen e Gestalt) para garantir usabilidade e clareza visual.

Componentes principais:

- Filtros interativos: categoria, mês e produto
- KPIs em destaque: mês com maior volume e total de vendas
- Tabela detalhada: produto, categoria, quantidade e valores
- Gráfico de área temporal: evolução mensal comparativa entre categorias
- Cards de extremos: produto mais vendido vs menos vendido

Design aplicado:

- Heurísticas de Nielsen: visibilidade de status, design minimalista, consistência visual
- Princípios de Gestalt: proximidade (agrupamento lógico), semelhança (paleta coesa), continuidade (fluxo visual)
- Paleta verde profissional com hierarquia clara de informação

---

## Insights & Tomada de Decisão

O dashboard permite identificar:

- Categoria com maior faturamento vs maior volume
- Produto destaque
- Produto com baixa performance
- Sazonalidade
- Padrões temporais para planejamento de estoque

Decisões estratégicas:

- Aumentar estoque de produtos de alta demanda (Monitor)
- Criar campanhas direcionadas para produtos de baixa performance (Smartphone)
- Ajustar estratégia de vendas baseada em sazonalidade identificada
- Balancear portfólio entre volume (Acessórios) e valor (Eletrônicos)

1. Faça o download da base de dados original
2. Abra o arquivo `dash_vendas_ti.pbix` no Power BI
3. Caso necessário, abra o editor do Power Query
4. Edite a primeira etapa de edição (Fonte) e selecione o caminho para a base baixada
